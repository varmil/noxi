# 自動 Lint 修正

## 作業完了後の必須ステップ

すべてのコード変更作業が完了した後、以下のステップを**必ず**実行してください：

### 1. Lint チェックの実行

```bash
npm run check
```

### 2. check エラーがある場合の対応

- エラーが検出された場合は、自動修正を試行：

```bash
npm run check -- --fix
```

### 3. 手動修正が必要な場合

- 自動修正できないエラーについては、エラーメッセージを確認して手動で修正
- 修正後、再度`npm run check`を実行して全エラーが解消されることを確認

### 4. 完了確認

- Lint エラーが 0 件になるまで上記プロセスを繰り返す
- 「All files pass linting」または類似のメッセージが表示されることを確認

## 実行タイミング

- リファクタリング作業後
- 新機能の実装完了後

## 注意事項

- このステップは作業の最終段階で実行し、ユーザーに作業完了を報告する前に必ず完了させる
- Lint エラーが残っている状態で作業完了とは見なさない
- エラーが解消できない場合は、その旨をユーザーに報告し、解決方法を提案する
