## 作業完了後の必須ステップ

すべてのコード変更作業が完了した後、以下のステップを**必ず**実行してください：

### 1. Type チェックの実行

```bash
npm run type-check
```

`/web` `/backend` それぞれにスクリプトがあります。作業で修正したファイルに応じて判断してください

### 2. エラーがある場合の対応

- エラーが検出された場合は、修正を試行：
- 修正後、再度`npm run type-check`を実行して全エラーが解消されることを確認

### 3. 完了確認

- エラーが 0 件になるまで上記プロセスを繰り返す

## 実行タイミング

- リファクタリング作業後
- 新機能の実装完了後

## 注意事項

- このステップは作業の最終段階で実行し、ユーザーに作業完了を報告する前に必ず完了させる
- エラーが残っている状態で作業完了とは見なさない
- エラーが解消できない場合は、その旨をユーザーに報告し、解決方法を提案する
